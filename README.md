# 1.项目描述
使用create-react-app搭建的一个实用的项目脚手架，使用的技术栈如下：
1.react
2.redux
3.express
4.sass
5.webpack
6.react-router
7.react-router-redux
8.axios

目前react版本为15.4.2，会基于项目进行不定时更新。

# 2.项目使用
### 第一步
克隆到本地

`git clone https://github.com/shifeng1993/react-start.git yourappname`

### 第二步
进入你的项目文件并安装依赖
`cd yourappname`

`npm install`
如果node-sass安装报错，需再次安装 `npm install node-sass`

### 第三步

使用`npm start` 开启项目
